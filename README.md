# Avinash Kumar Portfolio

A modern, responsive personal portfolio website for **Avinash Kumar**, Senior Software Engineer and IFS ERP Consultant.  
The portfolio highlights professional experience, technical skills, certifications, education, and contact details in a clean graphical layout.

## Live Website

[https://avink-dev.github.io](https://avink-dev.github.io)

## About

This portfolio is designed to present my experience in IFS ERP development, customization, integration, reporting, packaging, deployment, and consulting.  
It focuses on my work across IFS Apps 8, 9, 10, and IFS Cloud, along with experience in Boomi integrations, web services, PL/SQL, and enterprise application support.

## Features

- Responsive design for desktop, tablet, and mobile
- Graphical hero section with animated visual elements
- Career timeline for professional experience
- Skill cards and animated skill bars
- Sections for certifications, education, and recognition
- Contact section with email and GitHub profile
- Smooth scrolling and lightweight JavaScript interactions
- Static website suitable for GitHub Pages hosting

## Tech Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Portfolio Sections

- **Home** – Introduction and professional summary
- **About** – Overview of consulting and engineering experience
- **Impact** – Delivery approach and project workflow
- **Experience** – Professional journey across Xitricon, Tietoevry, and Capgemini
- **Skills** – IFS ERP, integrations, reporting, deployment, and programming skills
- **Certifications & Education** – Certifications, academic background, and awards
- **Contact** – Email, location, and GitHub profile
